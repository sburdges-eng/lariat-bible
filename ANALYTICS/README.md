# Analytics Directory

## Reports & Metrics

This directory contains templates and tools for tracking business performance, analyzing trends, and making data-driven decisions.

## Files (Templates)

### Monthly_Revenue_Report.xlsx
**Purpose**: Track monthly revenue and compare to targets

**Contains**:
- Event-by-event revenue breakdown
- Month-over-month comparisons
- Year-to-date totals
- Revenue by event type (wedding, corporate, private)
- Seasonal trend analysis

**Update Frequency**: Monthly (first week of following month)

---

### Food_Cost_Analysis.xlsx
**Purpose**: Monitor food costs and maintain target margins

**Contains**:
- Food cost per event
- Food cost percentage calculations
- Comparison to 30-35% target
- Ingredient cost trends
- Vendor pricing comparisons
- Waste tracking

**Update Frequency**: Per event, compiled monthly

---

### Client_Satisfaction_Tracker.xlsx
**Purpose**: Track client feedback and satisfaction scores

**Contains**:
- Client satisfaction ratings (1-5 scale)
- Feedback comments
- Repeat client tracking
- Referral sources
- Issue resolution notes
- Net Promoter Score (NPS)

**Update Frequency**: After each event

---

### Menu_Item_Popularity.xlsx
**Purpose**: Analyze which menu items are most popular

**Contains**:
- Order frequency by item
- Revenue per menu item
- Profit margin by item
- Seasonal popularity trends
- Client preferences
- Items to promote/retire

**Update Frequency**: Monthly review

---

## Key Performance Indicators (KPIs)

### Financial Metrics
- **Event Profitability**: Target 45% margin
- **Food Cost %**: Target 30-35%
- **Average Event Revenue**: Track and grow
- **Monthly Revenue**: Compare to goals
- **Year-over-Year Growth**: Measure expansion

### Operational Metrics
- **Prep Time Accuracy**: Actual vs. estimated
- **Ingredient Waste %**: Target <5%
- **On-Time Completion**: Event start time adherence
- **Staff Efficiency**: Labor hours per event
- **Order Accuracy**: Ingredient ordering precision

### Client Metrics
- **Client Satisfaction**: Target 4.5/5 average
- **Repeat Client Rate**: Track percentage
- **Referral Rate**: New clients from referrals
- **Booking Lead Time**: Average days in advance
- **Cancellation Rate**: Track and minimize

## How to Use Analytics

### Weekly Review
- Review upcoming events and revenue projections
- Check any pending client feedback
- Monitor prep time estimates vs. actuals

### Monthly Analysis
1. **Revenue Review**:
   - Total revenue vs. target
   - Number of events vs. capacity
   - Average revenue per event
   - Revenue by event type

2. **Cost Analysis**:
   - Overall food cost percentage
   - Vendor pricing changes
   - Waste and loss factors
   - Labor cost per event

3. **Quality Review**:
   - Client satisfaction scores
   - Any complaints or issues
   - Repeat vs. new client ratio
   - Referral sources

4. **Menu Analysis**:
   - Top performing items
   - Underperforming items
   - Seasonal trends
   - Pricing optimization opportunities

### Quarterly Strategy
- Review 3-month trends
- Adjust pricing if needed
- Update menu offerings
- Refine operational processes
- Set next quarter goals

### Annual Planning
- Year-end comprehensive review
- Set annual revenue targets
- Plan menu changes
- Evaluate vendor relationships
- Budget for equipment/improvements

## Data Sources

### From Active Events
- Invoice totals (revenue)
- Guest counts
- Menu item selections
- Ingredient costs (from orders)
- Prep time logs

### From Archive
- Historical event data
- Seasonal patterns
- Client history
- Successful event examples

### External Data
- Vendor price lists
- Market rates for catering
- Industry benchmarks
- Economic indicators

## Reporting Schedule

### Daily
- Update active event status
- Log any client communications
- Note any operational issues

### Weekly
- Revenue forecast for upcoming events
- Prep schedule review
- Ingredient inventory check

### Monthly
- Complete all four analytics templates
- Generate summary report
- Review KPIs against targets
- Identify action items

### Quarterly
- Trend analysis (3-month rolling)
- Strategic planning session
- Goal adjustment if needed
- Process improvement initiatives

### Annually
- Comprehensive year review
- Annual report for ownership
- Next year planning and budgeting
- System improvements roadmap

## Best Practices

1. **Consistency**: Update metrics on schedule
2. **Accuracy**: Double-check data entry
3. **Timeliness**: Enter data soon after events
4. **Completeness**: Don't skip data points
5. **Action-Oriented**: Use insights to improve
6. **Documentation**: Note reasons for anomalies
7. **Trends**: Look for patterns over time
8. **Benchmarking**: Compare to industry standards

## Common Insights

### High Food Cost %
Potential causes:
- Ingredient waste
- Recipe scaling errors
- Vendor pricing increases
- Theft or spoilage
- Portion control issues

### Low Client Satisfaction
Potential causes:
- Service timing issues
- Food quality problems
- Communication gaps
- Staff performance
- Unmet expectations

### Declining Revenue
Potential causes:
- Seasonal slowdown
- Market competition
- Pricing issues
- Marketing gaps
- Client retention problems
